# [입력과 출력](https://docs.python.org/ko/3.12/tutorial/inputoutput.html#)
- open(file, mode = 'r', encoding = None)
    - file : 파일명 
    - mode : 텍스트 모드
    - encoding : 인코딩 방식 (일반적으로 utf8 활용)
```
f = open('hello.txt', 'r', encoding = 'utf8')
text = type(f.read())
print(text)

>> <class 'str>
```
